# CrowdControl - Verilog

A digital circuit designed to monitor occupancy in venues 
(malls, hospitals, events) using entry/exit sensors.

## Features
- Real-time people counter via entry/exit sensors
- Dynamic capacity calculation based on available space
- Zone-level crowding detection across 4 zones (A, B, C, D)
- 3-clock-cycle timer to prevent false crowding alerts

## Files
- `CrowdControl.v` — Main module
- `TB_CrowdControl.v` — Testbench with 7 test scenarios
- Waveform simulation screenshot included
